अच् सन्धि

1) अकः सवर्णे दीर्घः
(अकः सवर्णे अचि पूर्वपरयोः एकः दीर्घः)
पूर्व वर्णः   पर वर्णः   सन्धि
अ/ आ + अ/ आ = आ
इ/ ई + इ/ ई = ई
उ/ ऊ + उ/ ऊ = ऊ
ऋ/ ॠ/ ऌ + ऋ/ ॠ/ ऌ = ॠ
2) आद् गुणः
(आत् इकि पूर्वपरयोः एकः गुणः)
पूर्व वर्णः   पर वर्णः   सन्धि
अ/ आ + इ/ ई = ए
उ/ ऊ = ओ
ऋ/ ॠ = अर्
ऌ = अल्
3) वृद्धिरेचि
(आत् एचि पूर्वपरयोः एकः वृद्धिः)
पूर्व वर्णः पर वर्णः सन्धि
अ/ आ + ए/ ऐ = ऐ
ओ/ औ = औ
4) एङि पररूपम्

5) अतो गुणे
(अपदान्तात् अतः गुणे पूर्वपरयोः एकः पररूपम्)

6) इको यणचि
(इकः अचि यण्)
पूर्व वर्णः   पर वर्णः   सन्धि
इ/ ई + असवर्णः स्वरः = य् + असवर्णः स्वरः
उ/ ऊ + असवर्णः स्वरः = व् + असवर्णः स्वरः
ऋ/ ॠ + असवर्णः स्वरः = र् + असवर्णः स्वरः
ऌ + असवर्णः स्वरः = ल् + असवर्णः स्वरः
7) एचोऽयवायावः
(एचः अचि अयवायावः)
पूर्व वर्णः   पर वर्णः   सन्धि
ए + अक् = अय् + अक्
ओ + अक् = अव् + अक्
ऐ + अक् = आय् + अक्
औ + अक् = आव् + अक्
अक् => अ, इ, उ, ऋ, ऌ
